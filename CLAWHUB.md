## One-Line Pitch

The machine that builds skills — because making a skill from scratch shouldn't feel like reinventing the wheel every time.

---

## What It Does

Skill Factory is your skill-making buddy. You bring an idea (or a URL), it handles the rest:

- **Lock the scope** — 3 quick questions to make sure you're building the right thing
- **Gather the goods** — fetches source material automatically
- **Distill the signal** — turns noise into structured SKILL.md + references
- **Ship it** — packages everything into a versioned, GitHub-ready repo
- **Keep it fresh** — structured upgrade workflow so your skills don't rot at v1.0

No boilerplate templates. No "what now?" moments. Just say the word and it works.

---

## When to Use

- You have a skill idea but don't know how to start
- You want to turn scattered notes or bookmarks into a proper skill
- You need to bring an existing skill up to date with new sources
- You're preparing something for GitHub or ClawHub
- You have a skill sitting at v1.0 and want to grow it without messing up

---

## Quick Start

Just tell your Claw:

```
New skill: [name], [1-2 sentences], [source URL (optional)]
```

Example:
```
New skill: prompt-engineering, best practices from Anthropic and OpenAI
```

Or to upgrade an existing one:
```
Upgrade skill: [name], [what to add], [new source URL (optional)]
```
