## One-Line Pitch

The machine that builds skills — turn any idea into a polished, versioned, publishable OpenClaw skill.

---

## What It Does

Skill Factory is a **meta-skill** that automates the full lifecycle of skill creation:

- **Phase 0:** Lock scope with 3 questions (prevents over-engineering)
- **Phase 1:** Collect raw sources automatically
- **Phase 2:** Distill into structured SKILL.md + references
- **Phase 3:** Publish as versioned, standalone repo
- **Upgrade Workflow:** Inspect → Classify → Propose → Apply → Summary (1→N)

From fuzzy idea → GitHub-ready skill in ~10 minutes.

---

## When to Use

- You have a skill idea but don't know where to start
- You want to package scattered knowledge into a reusable skill
- You need to upgrade an existing skill with new sources
- You're preparing a skill for GitHub/ClawHub distribution
